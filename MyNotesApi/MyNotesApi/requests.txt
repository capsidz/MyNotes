1. Создание заметки
{
    "title": "Моя первая заметка",
    "content": "Это содержание моей первой тестовой заметки для проверки работы API"
}

2. Создание заметки с ошибкой валидации
{
    "title": "",
    "content": "коротко"
}


3. Получение заметок
/notes

4. Получение конкретной заметки
/notes/1

5. Получение заметки, которой нет 
/notes /999

6. Поиск по заголовку
/notes/search?title=покупок

7. Изменение заметок
/notes/1

{
    "title": "Обновленная первая заметка",
    "content": "Это новое содержание"
}

8. Частичное изменение
/notes/2
{
    "content": "Обновленный список покупок: молоко, хлеб, яйца, сыр, йогурт, фрукты"
}

9. Удаление заметки
/notes/2

10. Удаление уже удаленной 
/notes/2
